# Различные варианты реализации игры "Жизнь", придуманной Джоном Конвеем

Приведены различные подходы к реализации. 
В каталоге `plane` имитация жизни осуществляется на ограниченной плоскости. 
В каталоге `tor` используется тороидальное пространство. 
При этом в каждом случае реализованы по три варианта:

1. Используется традиционное процедурное программирования на языке C, при котором состояние клеток задается перечислимым типом `enum` (проекты `conway-life-c`, `conway-life-c-tor`).
2. Вместо перечислимого типа используется обобщение, расширяемое специализациями, которые имитируют эволюционно расширяемый перечислимый тип. При этом используются прямые преобразования ячеек поля с применением функций, `cmp_spec`, `init_spec`, имитирующих сравнение и явное изменение специализаций в процедурном стиле (проекты `conway-life-ppc-cmp`, `conway-life-ppc-cmp-tor`).
3. Для обработки альтернативных состояний используется процедурно-параметрический полиморфизм, при котором для каждой из альтернатив используется свой обработчик специализации (проекты `conway-life-ppc-poly`, `conway-life-ppc-poly-tor`).
